# web-template-webpack5

# Features

Templates for web development. It supports webpack5 and can use typescript and scss.

# Installation

```bash
npm install
```

# Usage

There is npm script.

```bash
    "build": "webpack",
    "watch": "webpack --watch",
    "start": "webpack serve"
```

```bash
$ npm watch
$ npm start
```

# Note

Doesn't work hot reload.
